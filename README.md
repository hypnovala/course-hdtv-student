# HDTV — Student Course Portal
course.hypnoticdreamtv.com

## Structure
- index.html         → main course page (deploy root)
- public/videos/     → drop mod1_videohdtv.mp4 … mod7_videohdtv.mp4 here
- public/images/     → instructor photos, banners
- public/documents/  → student guide & worksheet PDFs
- vercel.json        → tells Vercel to serve from repo root (.)

## Deploy
1. Push to GitHub
2. Import repo in Vercel dashboard
3. Framework: Other (no framework)
4. Output Directory: . (dot — repo root)
5. Assign domain: course.hypnoticdreamtv.com
